# Hotel Reservation System

A professional and interactive Java Swing application for managing hotel room availability and bookings.

## Features

- View all hotel rooms with their type, price, and availability status
- Book an available room by entering guest details and check-in/out dates
- View a list of all current bookings
- Cancel an existing booking, which automatically makes the room available again
- Uses a tabbed interface for a clean separation of concerns
- Simple data validation for booking details

## How to Run

1. Ensure you have a JDK (Java Development Kit) installed
2. Download the project as a ZIP file or clone it
3. Extract the files
4. Open a terminal or command prompt in the project's root folder
5. Compile the code: `javac -cp "lib\flatlaf-3.2.5.jar" -d src src\*.java`
6. Run the application: `java -cp "src;lib\flatlaf-3.2.5.jar" HotelSystemGUI`

## Technologies Used
